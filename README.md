# Запуск приложения

Для запуска FastAPI используется веб-сервер uvicorn. Команда для запуска выглядит так:

uvicorn app.main:app --reload

Ее необходимо запускать в командной строке, обязательно находясь в корневой директории проектаh

Далее создать базу данных с данными указанными в файле 
.env

После перейти http://127.0.0.1:8000/docs  и воспользоваться необходимыми эндпоинтами
