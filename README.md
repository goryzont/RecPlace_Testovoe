# Запуск приложения

Для запуска FastAPI используется веб-сервер uvicorn. Команда для запуска выглядит так:

uvicorn app.main:app --reload

Ее необходимо запускать в командной строке, обязательно находясь в корневой директории проекта

Далее необходимо создать базу данных с данными, указанными в файле 
.env

После перейти на http://127.0.0.1:8000/docs и воспользоваться необходимыми эндпоинтами

# Эндпоинты

# POST/auth/register и POST/auth/login

Эти эндпоинты отвечают за регистрацию и аутентификацию пользователя

# GET/auth/get_user

Этот эндпоинт возвращает данные о пользователе по его id

# POST/auth/request

Этот эндпоинт проверяет, аутентифицирован ли пользователь или нет,
И если он аутентифицирован, то пользователь может отправить запрос(вопрос)

# GET/auth/last_three

Этот эндпоинт возвращает последние 3 запроса(вопроса) пользователя, используя его id
